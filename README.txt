JQuery processing framework is found in src/org/exist/xquery/lib

To install:

cd into the tamboti directory and call ant; a file, library-1.0.xar, is created.

log in as admin to eXist, go to the Package Reposistory, choose library-1.0.xar and upload it.

click "Install" and access tamboti at <http://localhost:8080/exist/apps/library/>.
